# Cible

Folia vise le plus large public possible — du curieux débutant au professionnel expert.
Chaque profil dispose d'une **interface personnalisée** adaptée à ses besoins et son niveau.

## Profils

### 🌱 Le Débutant
- Vient de découvrir les plantes, cherche des conseils simples
- Besoins : identification facile, conseils d'entretien de base, encouragements
- Interface : simplifiée, visuelle, gamifiée
- Exemple : quelqu'un qui vient d'acheter sa première plante d'intérieur

### 🪴 Le Passionné
- Possède une collection, suit les tendances (plantes rares, urban jungle...)
- Besoins : carnet d'entretien, communauté, références d'espèces, achat/échange
- Interface : social en premier, gestion de collection, notifications
- Exemple : passionné Instagram avec 50+ plantes

### 🌿 Le Jardinier
- Orienté jardinage extérieur, légumes, fruits, arbustes, arbres
- Besoins : calendrier saisonnier, guides de plantation, intégration météo locale
- Interface : guidée par le calendrier, pratique, géo-consciente
- Exemple : quelqu'un avec un potager ou un petit verger

### 🔬 Le Botaniste / Expert
- Connaissance scientifique approfondie, orienté taxonomie
- Besoins : données taxonomiques complètes, journal d'observations, cartographie des espèces, discussions communautaires
- Interface : dense en données, nomenclature scientifique, outils de contribution
- Exemple : professeur de biologie, naturaliste amateur, botaniste professionnel

### 🏪 Le Professionnel
- Pépiniériste, paysagiste, fleuriste
- Besoins : gestion de catalogue, recommandations clients, fonctionnalités B2B
- Interface : orientée métier, outils en masse, intégrations partenaires
- Exemple : gérant d'une jardinerie, architecte paysagiste

## Plateforme

**Mobile-first** — l'app est conçue pour le smartphone comme device principal.
Le web est une surface secondaire pour les fonctionnalités riches en contenu ou orientées desktop.
