# Fonctionnalité — Catalogue de plantes

## Description
Base de données de référence des plantes, validée par des experts. Point d'entrée principal de l'application.

## Contenu d'une fiche plante
- Nom scientifique et noms communs
- Taxonomie complète (famille, genre, espèce)
- Photos de référence
- Conseils d'entretien (arrosage, lumière, humidité, température, sol)
- Caractéristiques (intérieur/extérieur, toxicité, comestible, animal-friendly)
- Aire de répartition géographique
- Niveau de difficulté
- Maladies et problèmes courants + traitements

## Comportement par profil
- **Débutant** : infos simplifiées, iconographie, conseils pratiques
- **Passionné** : infos complètes, galerie, espèces similaires
- **Jardinier** : calendrier saisonnier, compatibilité avec d'autres plantes
- **Expert** : nomenclature scientifique complète, données taxonomiques, sources

## MVP
- Fiches complètes avec contenu expert
- Recherche par nom, type, difficulté
- Filtres : intérieur/extérieur, toxicité, niveau d'entretien
