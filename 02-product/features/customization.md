# Fonctionnalité — Personnalisation de l'Interface

## Description
Chaque utilisateur configure son expérience Folia selon ses priorités et ses goûts.

## Dashboard configurable
- Widgets repositionnables par drag & drop
- Widgets disponibles :
  - Plante du jour
  - Prochains entretiens
  - Météo jardinage
  - Activité communauté
  - Mes dernières observations
  - Progression et grade
  - Défi en cours
- Sauvegarde de la configuration par profil

## Thèmes visuels
- Clair, sombre, automatique (selon l'heure)
- Botanique vintage — illustrations aquarelle, tons sépia
- Moderne minimaliste — épuré, focus sur les photos
- Nature immersive — fonds animés, textures organiques
- Thèmes saisonniers exclusifs (débloqués via les saisons de jeu)

## Mode focus
- Une seule plante à la fois, interface épurée sans distraction
- Idéal pour les sessions d'entretien concentrées
- Minuteur optionnel pour les tâches longues

## Notifications personnalisables
- Choisir exactement ce qu'on veut recevoir
- Heure de réception configurable par type de rappel
- Mode "Ne pas déranger" avec plages horaires
- Digest quotidien ou hebdomadaire en remplacement des notifications unitaires

## Langue et région
- Interface disponible en plusieurs langues
- Unités configurables : °C/°F, cm/inches, litres/gallons
- Format de date selon la région

## Accessibilité
- Taille de police ajustable
- Mode daltonien (deutéranopie, protanopie, tritanopie)
- Contrastes élevés
- Compatibilité lecteurs d'écran (VoiceOver, TalkBack)
- Réduction des animations pour les épileptiques

## MVP
- Thème clair / sombre (V1)
- Notifications personnalisables (V1)
- Dashboard configurable (V2)
- Thèmes visuels avancés (V2)
- Mode focus (V2)
