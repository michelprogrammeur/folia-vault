# Fonctionnalité — Intégrations Tierces

## Description
Folia s'intègre dans l'écosystème d'apps déjà utilisé par ses utilisateurs pour une expérience fluide et unifiée.

## Calendriers
- **Google Calendar / Apple Calendar** : sync automatique des rappels d'entretien
- Les événements apparaissent dans le calendrier habituel de l'utilisateur
- Titre clair : "Arroser le Ficus benjamina", "Engrais Monstera"
- Option : sync bidirectionnel (marquer comme fait dans le calendrier = fait dans Folia)

## Météo
- **Météo France, OpenWeatherMap, Weather API**
- Données météo locales pour adapter les rappels et les alertes
- Prévisions 7 jours intégrées dans le calendrier de jardinage
- Alertes gel, canicule, sécheresse automatiques

## Réseaux sociaux
- **Instagram** : import de photos depuis son feed directement dans sa collection
- **Pinterest** : sauvegarder des inspirations jardinage liées à des plantes du catalogue
- Partage facile de sa collection, ses badges, son rapport annuel sur tous les réseaux

## Apps santé
- **Apple Health / Google Fit** : temps de jardinage comptabilisé comme activité physique
- Partage optionnel des données bien-être

## Musique
- **Spotify / Apple Music** : playlists jardinage thématiques selon la saison ou l'activité
- "Playlist taille du dimanche matin", "Ambiance serre tropicale"
- Playlists curatorées par Folia et par la communauté

## Smart Home et IoT
- **Google Home / Amazon Alexa** : "Alexa, demande à Folia ce que j'ai à faire aujourd'hui"
- Capteurs de sol connectés (Xiaomi, Parrot Flower Power...)
- Automatisation : si humidité sol < X% → notification arrosage

## API publique Folia
- Accès aux données du catalogue pour les développeurs tiers
- Intégration possible dans des apps e-commerce, déco, jardinage
- Documentation publique et sandbox de test
- Modèle freemium : quota gratuit, abonnement pour usage intensif

## MVP
- Partage réseaux sociaux (V1)
- Météo locale pour les alertes (V1)
- Google / Apple Calendar sync (V2)
- Spotify playlists (V2)
- API publique (V3)
- IoT capteurs (V3)
