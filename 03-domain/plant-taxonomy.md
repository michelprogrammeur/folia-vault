# Taxonomie des Plantes

## La classification du vivant (rappel)

```
Règne      → Plantae
  Division   → ex: Magnoliophyta (plantes à fleurs)
    Classe     → ex: Magnoliopsida (dicotylédones)
      Ordre      → ex: Alismatales
        Famille    → ex: Araceae
          Genre      → ex: Monstera
            Espèce     → ex: Monstera deliciosa
              Sous-espèce / Variété / Cultivar / Hybride
```

---

## Niveaux taxonomiques dans Folia

### Famille (`Family`)
- Regroupe des genres partageant des caractéristiques communes
- Nom toujours en latin, se termine généralement en `-aceae`
- Ex: Araceae, Rosaceae, Orchidaceae, Poaceae, Cactaceae
- Données stockées : nom latin, nom commun, description, nombre de genres, répartition géographique

### Genre (`Genus`)
- Subdivision d'une famille
- Nom latin avec majuscule, toujours en italique dans les textes
- Ex: *Monstera*, *Ficus*, *Rosa*, *Aloe*
- Données stockées : nom latin, famille parente, description, nombre d'espèces, origine géographique

### Espèce (`Species`)
- Unité de base — organismes capables de se reproduire entre eux
- Nommée par un **binôme latin** : Genre + épithète spécifique (en minuscule)
- Ex: *Monstera deliciosa*, *Ficus benjamina*, *Rosa canina*
- Toujours en italique dans les textes scientifiques
- Données stockées : voir ci-dessous

### Sous-espèce / Variété (`Subspecies / Variety`)
- Subdivision d'une espèce avec des caractéristiques légèrement différentes
- Notation : *Espèce* subsp. *nom* ou *Espèce* var. *nom*
- Ex: *Monstera deliciosa* var. *borsigiana*

### Cultivar
- Variété sélectionnée et maintenue par l'homme (culture)
- Notation : *Espèce* 'Nom du cultivar' (entre guillemets simples, sans italique)
- Ex: *Monstera deliciosa* 'Thai Constellation', *Rosa* 'Peace'
- Ne se reproduit pas fidèlement par graines — multiplication végétative

### Hybride
- Croisement entre deux espèces ou genres différents
- Notation avec × : *Genre* × *épithète* ou *Genre* ×*hybridename*
- Ex: *Fatshedera lizei* (hybride entre Fatsia et Hedera)

---

## Nomenclature binomiale (règles de Linné)

- Toujours en latin ou latinisé
- Genre avec **majuscule**, épithète en **minuscule**
- Toujours en **italique** dans les textes
- Peut être suivi du nom de l'auteur (abrégé) : *Monstera deliciosa* Liebm.
- En cas de révision taxonomique, l'auteur original est entre parenthèses : *Epipremnum aureum* (Linden & André) G.S.Bunting

## Données d'une fiche espèce dans Folia

```
Identité
  - Nom scientifique (binôme complet)
  - Auteur et année de description
  - Synonymes connus
  - Noms vernaculaires (par langue)
  - Famille / Genre / Espèce

Origine et répartition
  - Aire de répartition naturelle (pays, continents)
  - Habitat naturel (forêt tropicale, désert, zone tempérée...)
  - Altitude (min/max)
  - Statut de conservation (UICN : LC, NT, VU, EN, CR, EW, EX)

Morphologie
  - Type de plante (herbacée, arbuste, arbre, liane, succulente...)
  - Hauteur et largeur maximales
  - Description des feuilles (forme, couleur, texture, taille)
  - Description des fleurs (couleur, période de floraison, inflorescence)
  - Description des fruits et graines
  - Système racinaire

Entretien
  - Niveau de difficulté (facile, moyen, difficile)
  - Lumière (ombre, mi-ombre, plein soleil, indirect)
  - Arrosage (fréquence, quantité, sensibilité)
  - Humidité (faible, moyenne, élevée)
  - Température (min/max, sensibilité au gel)
  - Sol (type, pH, drainage)
  - Engrais (type, fréquence, saison)
  - Rempotage (fréquence, période)

Caractéristiques
  - Intérieur / Extérieur / Les deux
  - Toxicité (humains, chiens, chats, autres animaux)
  - Comestible (fruits, feuilles, racines...)
  - Mellifère (attractivité pour les pollinisateurs)
  - Dépolluante (polluants absorbés)
  - Invasive dans certaines régions

Propagation
  - Méthodes disponibles (bouturage, semis, division, marcottage...)
  - Difficulté par méthode
  - Meilleure période
  - Taux de réussite moyen

Médias
  - Photos de référence (feuilles, fleurs, port général)
  - Modèle 3D (si disponible)
  - Vidéos d'entretien
```

## Exemple bout en bout — *Monstera deliciosa* 'Thai Constellation'

```
Règne      → Plantae
  Division   → Tracheophyta (plantes vasculaires)
    Classe     → Liliopsida (monocotylédones)
      Ordre      → Alismatales
        Famille    → Araceae (famille des arums)
          Genre      → Monstera Adans. (1763)
            Espèce     → Monstera deliciosa Liebm. (1849)
              Cultivar   → Monstera deliciosa 'Thai Constellation'
```

**Auteur** : Friedrich Michael Liebmann (Liebm.) — botaniste danois, a décrit l'espèce en 1849
**Synonymes** : *Philodendron pertusum* Kunth & C.D.Bouché (ancien nom, toujours utilisé en horticulture)
**Nom vernaculaire** : Monstera, Plante fenêtrée, Faux Philodendron, Délicieuse de Liebmann
**Cultivar 'Thai Constellation'** : sélection à feuilles panachées crème, multiplication uniquement végétative (tissue culture)
**Statut UICN** : LC (Préoccupation mineure)
**Origine** : forêts tropicales humides du Mexique et d'Amérique centrale

## Types de végétation et durée de vie

### Par durée de vie
| Type | Définition | Exemples |
|---|---|---|
| **Annuelle** | Cycle complet en une saison, meurt après la graine | Tournesol, basilic, capucine |
| **Bisannuelle** | Végétation la 1ère année, floraison la 2ème | Carotte, persil, digitale |
| **Vivace** | Repousse chaque année, partie aérienne peut mourir en hiver | Lavande, hostas, graminées |
| **Persistante** | Garde ses feuilles toute l'année | Ficus, monstera, laurier |
| **Caduque** | Perd ses feuilles en automne/hiver | Chêne, hêtre, rosier |

### Par port
| Type | Définition | Exemples |
|---|---|---|
| **Herbacée** | Tige non ligneuse, souple | Basilic, fougère, ciboulette |
| **Arbuste** | Ligneux, plusieurs tiges, < 7m | Forsythia, lilas, buis |
| **Arbre** | Ligneux, tronc unique, > 7m | Chêne, pommier, palmier |
| **Liane** | S'accroche ou grimpe | Pothos, lierre, glycine |
| **Succulente** | Stocke l'eau dans ses tissus | Aloe, cactus, sedum |
| **Épiphyte** | Vit sur d'autres plantes sans les parasiter | Orchidées, broméliacées, tillandsia |
| **Aquatique** | Vit dans ou près de l'eau | Nénuphar, papyrus, lotus |

### Par usage
| Usage | Exemples |
|---|---|
| **Ornementale** | Monstera, rose, dahlia |
| **Comestible** | Tomate, pomme, salade |
| **Aromatique** | Thym, menthe, basilic |
| **Médicinale** | Aloe vera, échinacée, valériane |
| **Mellifère** | Lavande, bourrache, phacélie |
| **Dépolluante** | Spathiphyllum, chlorophytum, pothos |
| **Industrielle** | Lin, coton, bambou |

## Statuts de conservation UICN

| Code | Statut | Description |
|---|---|---|
| EX | Éteint | Plus aucun individu connu |
| EW | Éteint à l'état sauvage | Survit uniquement en captivité |
| CR | En danger critique | Risque extrêmement élevé d'extinction |
| EN | En danger | Risque très élevé d'extinction |
| VU | Vulnérable | Risque élevé d'extinction |
| NT | Quasi menacé | Proche du seuil de vulnérabilité |
| LC | Préoccupation mineure | Population stable |
| DD | Données insuffisantes | Pas assez d'informations |
